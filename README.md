# CodeGenius - Sistema de Quiz

Aplicação web completa para criação e execução de quizzes educacionais, permitindo que usuários criem perguntas, respondam quizzes e acompanhem seu desempenho.

---

## Funcionalidades

* Cadastro e autenticação de usuários (login/logout)
* Criação e gerenciamento de quizzes
* Execução de quizzes em tempo real
* Sistema de pontuação e ranking
* Armazenamento de resultados
* Interface web interativa

---

## Tecnologias utilizadas

* Node.js
* Express
* MySQL
* HTML, CSS, JavaScript

---

## Arquitetura

O projeto foi estruturado de forma modular, separando responsabilidades entre rotas, lógica de negócio e acesso a dados, facilitando manutenção e escalabilidade.

---

## Como executar

Clone o repositório:

```bash
git clone https://github.com/Joaogrlobo/CodeGenius.git
```

Acesse a pasta do projeto:

```bash
cd CodeGenius
```

Instale as dependências:

```bash
npm install
```

Configure o banco de dados MySQL com as credenciais necessárias.

Execute a aplicação:

```bash
npm start
```
